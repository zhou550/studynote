## Flask

### 初始化参数

* `import_name`
* static_url_path
* static_folder
* template_folder

### 工程配置参数

* app.config.from_object(配置对象)
   * 继承 —> 优点 复用
   * 敏感数据暴露  缺点
* app.config.from_pyfile(配置文件)
   * 优点 -> 独立文件 保护敏感数据
   * 缺点 -> 不能继承 文件路径固定 不灵活
* app.config.from_envvar('环境变量名')
   * 优点 -> 独立文件 保护敏感数据  文件路径不固定 灵活
   * 缺点-> 不方便 要记得设置环境量
   * 设置环境变量
      * 终端 export
      * pycharm 设置
